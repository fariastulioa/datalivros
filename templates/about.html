Sobre o projeto

O Banco de Livros registra livos lidos, armazenando seus dados e permitindo que os mesmos sejam visualizados ou atualizados.
O funcionamento do sistema é regido por código Python, baseado no framework Flask para aplicações web.
Por meio do Python, o usuário interage com o banco de dados SQLITE diretamente do navegador, em
páginas web construidas com html e Bootstrap. A aplicação Python também permite manipular
os dados em objetos DataFrame (Pandas) para análises estatísticas e visualizações de dados.
